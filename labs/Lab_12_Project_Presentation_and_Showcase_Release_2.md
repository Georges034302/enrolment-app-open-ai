# Lab 12 - Project Presentation and Showcase (Release 2)

## Lab Type

Showcase lab (execution and evidence only, no new teaching content).

Deployment scope includes:
- Local execution using Docker Compose with AI-Mode, MCP, RAG and Multi-Agent services enabled.
- Cloud deployment to Microsoft Azure or AWS with AI-Mode, Ollama and approved LLMs enabled. MCP, RAG and Multi-Agent services remain disabled in the cloud.

## Goal

Prepare and present the completed Release 2 Agentic AI application using seven evidence trees:

1. Report Tree
2. Working Software Tree
3. Multi-Agent Workflow Tree
4. Software Testing Tree
5. DevOps and CI/CD Tree
6. Local & Cloud Deployment Tree
7. Contribution Tree

Each student must present their assigned frontend, backend/API and SQLite database microservice together with local deployment, cloud deployment and Multi-Agent integration evidence.
