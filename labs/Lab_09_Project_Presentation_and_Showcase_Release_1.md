# Lab 09 - Project Presentation and Showcase (Release 1)

## Lab Type

Showcase lab (execution and evidence only, no new teaching content).

Deployment scope for this showcase is local execution of the integrated Release 1 application using Docker Compose. Cloud deployment is out of scope for Release 1.

## Goal

Prepare and present the Release 1 integrated Agentic AI software application using six evidence trees:

1. Report Tree
2. Working Software Tree
3. Agentic AI Workflow Tree
4. DevOps and CI/CD Tree
5. MCP & RAG Tree
6. Contribution Tree

Each student must present their assigned frontend, backend/API, and SQLite database microservice extension as part of the integrated Release 1 application.

This showcase extends Release 0 by integrating MCP, RAG and grounded AI responses.
